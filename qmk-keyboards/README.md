# How to use QMK keyboard
Use a chromium based browser and open [usevia.app](https://usevia.app/)

## Tap for ESC and hold for left ctrl

```
MT(MOD_LCTL,KC_ESC)
```

## Tap for key command and hold for layer

```
LT(1,KC_TAB)

LT(2,KC_SPC)
```
## Fn key hold for toggle

```
MO(1)
```
